## 简介

FizeDatabase 是一个全功能、易于扩展的 PHP 数据库类库，提供了统一的数据库访问接口，支持多种数据库类型（MySQL、PostgreSQL、SQLite、Oracle、SQL Server、Access）和连接模式（PDO、MySQLi、ODBC、ADODB）。它采用面向对象的设计，通过工厂模式和中间件机制实现了良好的可扩展性和易用性。

## 安装步骤

### 通过 Composer 安装

FizeDatabase 可以通过 Composer 包管理器轻松安装：

```bash
composer require fize/database
```

### 验证安装

安装完成后，可以通过以下方式验证安装是否成功：

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

echo "FizeDatabase 安装成功！";
```

## 环境要求

### PHP 版本要求

- **最低要求**: PHP 7.1.0 或更高版本
- **推荐版本**: PHP 7.2.0 或更高版本

### 必需的 PHP 扩展

FizeDatabase 会根据不同的数据库类型和连接模式自动检测所需的 PHP 扩展：

- **PDO 扩展**: 所有数据库驱动的基础
- **MySQLi 扩展**: MySQL 连接支持
- **ODBC 扩展**: ODBC 兼容数据库支持

### 可选的 PHP 扩展

为了获得更好的性能和更多功能，建议安装以下扩展：

- **iconv 扩展**: 字符编码转换支持
- **oci8 扩展**: Oracle 数据库支持
- **pgsql 扩展**: PostgreSQL 数据库支持
- **sqlite3 扩展**: SQLite 数据库支持
- **sqlsrv 扩展**: SQL Server 数据库支持

## 基本配置

### 数据库连接配置

FizeDatabase 支持多种数据库类型和连接模式。以下是基本的连接配置示例：

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

// MySQL 连接配置
$config = [
    'host'     => 'localhost',    // 数据库主机地址
    'user'     => 'username',     // 数据库用户名
    'password' => 'password',     // 数据库密码
    'dbname'   => 'database_name' // 数据库名称
];

// 创建数据库连接
new Db('mysql', $config, 'pdo');
```

### 连接参数详解

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| host | string | 是 | - | 数据库服务器地址 |
| user | string | 是 | - | 数据库用户名 |
| password | string | 是 | - | 数据库密码 |
| dbname | string | 是 | - | 数据库名称 |
| port | int | 否 | 空 | 数据库端口号 |
| charset | string | 否 | utf8 | 字符集设置 |
| prefix | string | 否 | 空 | 表前缀 |
| opts | array | 否 | 空数组 | PDO 连接选项 |

## 5分钟快速示例

### 第一步：创建数据库连接

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

// 数据库配置
$config = [
    'host'     => 'localhost',
    'user'     => 'root',
    'password' => '123456',
    'dbname'   => 'test_db'
];

// 创建数据库连接
new Db('mysql', $config, 'pdo');

echo "数据库连接成功！\n";
```

### 第二步：创建测试表

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

// 创建用户表（如果不存在）
$sql = "CREATE TABLE IF NOT EXISTS users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
)";

Db::query($sql);
echo "用户表创建成功！\n";
```

### 第三步：基础 CRUD 操作

#### 插入数据

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

// 插入单条数据
$insertData = [
    'name'  => '张三',
    'email' => 'zhangsan@example.com'
];

$result = Db::table('users')->insert($insertData);
echo "插入成功，受影响行数：" . $result . "\n";

// 插入多条数据
$batchData = [
    ['name' => '李四', 'email' => 'lisi@example.com'],
    ['name' => '王五', 'email' => 'wangwu@example.com']
];

// 使用批量插入（如果数据库驱动支持）
// $result = Db::table('users')->insertAll($batchData);
```

#### 查询数据

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

// 查询所有用户
$users = Db::table('users')->select();
echo "所有用户：\n";
foreach ($users as $user) {
    echo "ID: {$user['id']}, 姓名: {$user['name']}, 邮箱: {$user['email']}\n";
}

// 条件查询
$user = Db::table('users')
    ->where(['name' => '张三'])
    ->find();
    
if ($user) {
    echo "找到用户：{$user['name']} - {$user['email']}\n";
}

// 带条件的查询
$users = Db::table('users')
    ->where(['email' => ['LIKE', '%example%']])
    ->select();
```

#### 更新数据

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

// 更新用户信息
$result = Db::table('users')
    ->where(['name' => '张三'])
    ->update(['email' => 'newemail@example.com']);

echo "更新成功，受影响行数：" . $result . "\n";
```

#### 删除数据

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

// 删除用户
$result = Db::table('users')
    ->where(['name' => '张三'])
    ->delete();

echo "删除成功，受影响行数：" . $result . "\n";
```

### 第四步：高级查询示例

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

// 复杂条件查询
$users = Db::table('users')
    ->where([
        'name' => ['LIKE', '%张%'],
        'created_at' => ['>=', '2023-01-01']
    ])
    ->order(['created_at' => 'DESC'])
    ->limit(10)
    ->select();

// 分页查询
$page = 1;
$size = 10;
$users = Db::table('users')
    ->page($page, $size)
    ->order(['id' => 'ASC'])
    ->select();

// 统计查询
$count = Db::table('users')->count();
echo "总用户数：" . $count . "\n";
```

### 第五步：事务处理

```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

try {
    // 开始事务
    Db::startTrans();
    
    // 执行多个数据库操作
    Db::table('users')->insert(['name' => '事务用户1', 'email' => 'user1@example.com']);
    Db::table('users')->insert(['name' => '事务用户2', 'email' => 'user2@example.com']);
    
    // 提交事务
    Db::commit();
    echo "事务提交成功！\n";
    
} catch (Exception $e) {
    // 回滚事务
    Db::rollback();
    echo "事务回滚，错误：" . $e->getMessage() . "\n";
}
```

## 常见问题解答

### 安装问题

**问题：Composer 安装失败**
- 检查网络连接是否正常
- 尝试使用国内镜像源：`composer config repositories.packagist https://mirrors.aliyun.com/composer/`
- 清理 Composer 缓存：`composer clear-cache`

**问题：PHP 版本过低**
- 升级到 PHP 7.1.0 或更高版本
- 检查 `php -v` 输出确认版本

**问题：缺少 PHP 扩展**
- 安装必需的扩展：`sudo apt-get install php-mysqli php-pdo php-odbc`
- 重启 Web 服务器使扩展生效

### 连接问题

**问题：数据库连接失败**
```php
<?php
require_once "vendor/autoload.php";

use Fize\Database\Db;

$config = [
    'host'     => 'localhost',
    'user'     => 'root',
    'password' => '123456',
    'dbname'   => 'test_db',
    'port'     => 3306
];

try {
    new Db('mysql', $config, 'pdo');
    echo "连接成功！";
} catch (Exception $e) {
    echo "连接失败：" . $e->getMessage();
}
```

**问题：字符集设置问题**
- 确保数据库字符集与 PHP 设置一致
- 在配置中设置正确的 charset 参数

### 查询问题

**问题：SQL 注入防护**
FizeDatabase 自动处理 SQL 注入防护，始终使用参数绑定：
```php
// 正确的方式
Db::table('users')->where(['name' => $userName])->find();

// 错误的方式（不推荐）
$sql = "SELECT * FROM users WHERE name = '" . $userName . "'";
```

**问题：查询性能优化**
- 使用适当的索引
- 避免 SELECT *
- 合理使用 LIMIT
- 使用缓存机制
